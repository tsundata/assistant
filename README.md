Assistant Bot is a workflow engine for chatbot

## Features

- Chat bot
- Message Publish/Subscribe Hub
- Message Cron, Trigger, Task, Pipeline
- Workflow Action ([Syntax](./doc/action_syntax.md))

## Requirements

This project requires Go 1.16 or newer

## Installation

1. Install MySQL, Redis, influx, jaeger, nats, consul

2. Import sql files

3. Import Configuration to consul

4. Set Environment
   
   See [doc/env.md](/doc/env.md)

5. Build binary
   ```
   task build
   ```

6. Run App binary
